# Tableau Dashboard

## English

### Data:

- record_id, unique video number
- region, country
- trending_date, video date
- category_title, video group type
- videos_count, number of videos

### Objective :

1. Create a tableau dashboard based on a predetermined format.
2. Make a presentation.
